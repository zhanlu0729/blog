## 安装
### 
redis-3.2.2

### 单机模式
1. **下载** wget http://download.redis.io/releases/redis-3.2.2.tar.gz
2. **解压** tar -zxvf redis-3.2.2.tar.gz
3. **建立软链接,便于升级** ln -s redis-3.2.2 redis
4. **配置** ./configure
5. **安装** make && make install
6. **启动** ./redis-server
